# SetPowerSaver — Power scripts for Windows 11 (developer-friendly)

**EN**  
Two Windows PowerShell 5.x scripts:
- `SetPowerSaver.ps1` — Enable Hibernate, switch to Power Saver, set display/sleep/hibernate timeouts.
- `RunKeepAliveMaxPerf.ps1` — Prevent idle sleep/hibernate (Win32 heartbeat) and temporarily switch to High performance. Closing the window restores your original plan.

**TH**  
สคริปต์ PowerShell 5.x สองตัว:
- `SetPowerSaver.ps1` — เปิด Hibernate, สลับ Power Saver, ตั้งเวลา ปิดจอ/สลีป/ไฮเบอร์เนต
- `RunKeepAliveMaxPerf.ps1` — กันเครื่องไม่ให้หลับ/ไฮเบอร์เนตจาก idle และสลับ High performance ชั่วคราว ปิดหน้าต่างจะคืนค่าเดิม
